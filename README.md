# Peaks nextjs static client app

- Version: 0.0.0
- Last changes timestamp: 2023.01.26, 02:26

## Install

Install all required node dependencies:

```
npm i
```

## Start dev server

Start dev server (locate in browser with `http://localhost:3000/agreement`):

```
npm run dev
```

## Make build

```
npm run build
```
